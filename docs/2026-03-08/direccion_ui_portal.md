# Direccion UI Portal

## Direccion visual

Inspiracion:

- rojo, blanco y negro
- interfaz sobria
- lectura clara
- sensacion corporativa

## Regla visual

- fondo claro o blanco hueso
- bloques blancos con sombra corta
- rojo principal para acentos y llamadas a la accion
- negro para titulos y texto importante
- gris solo para texto secundario

## Comportamiento esperado

- dashboard simple
- acceso rapido a apps
- perfil visible
- acciones principales arriba
- tarjetas faciles de escanear

## Objetivo UX

- que el usuario entre y entienda en segundos:
  - quien es
  - donde esta
  - que apps tiene
  - como salir
