# Politica de Retencion de Logs

- Logs de aplicacion: 30 a 90 dias segun volumen y costo.
- Logs de seguridad: minimo 180 dias; ideal 365 dias.
- Incidentes de aplicacion: conservar mas tiempo que logs operativos.
- Metricas agregadas de negocio: largo plazo para tendencias.
- Logs con secretos o datos sensibles no deben persistirse completos.
